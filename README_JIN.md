## 🚀 核心项目 (Core Projects)

### 1. LACM- Enhanced EfficientNet Elevating Precision and CPU Efficiency in Agricultural Pest and Disease Recognition

- **项目简介**：针对 EfficientNet 中 SE 模块空间局部建模缺失及边缘端精度‑效率悖论，提出 LACM 动态局部感知混合模块，实现空间‑通道协同与轻量化推理。
- **核心工作**：
  - 基于 PyTorch 重构 EfficientNet 并引入 LACM 模块，实现动态局部卷积核生成与通道分组融合
  - 整合 6 大类公开农业病虫害数据，经去重与分层采样构建 7.3 万样本混合数据集
  - 设计消融实验并调优超参数，模型准确率提升至 **92.27% (+2.24%)**，CPU 端推理延迟降至 **55.1ms（加速 16%）**，FLOPs 压缩至 **0.272G**

### 2. 智膳热鉴 —— 多模态大模型热量识别系统

- **项目简介**：针对经验估算热量误差高及视觉大模型的“领域知识幻觉”问题，基于 Qwen-VL-Max 多模态大模型与 RAG 检索增强生成，构建 B/S 架构热量识别系统，实现食物图像智能解析与热量精算。
- **核心工作**：
  - 基于 SpringBoot + MySQL 构建服务层与数据持久层
  - 采用轻量级启发式 RAG 引擎，设计多维权重打分算法，实现垂直领域专家知识的毫秒级召回与 Prompt 动态组装

### 3. CoT-RolePlay —— 异构秩分配与焦点损失的推理增强微调策略

- **项目简介**：针对 1.5B 参数大模型微调时“小样本易过拟合”及“高频虚词梯度过载”问题，以 DeepSeek-R1-Distill 为基座，在受限算力下构建可调参数约 **1%** 的角色对齐与认知解耦微调策略。
- **核心工作**：
  - 利用知识蒸馏自主合成 **1.8 万条**双段式 CoT 语料
  - 引入异构秩分配：MLP 层（r=32）与 Attention 层（r=8）差异化旁路矩阵注入
  - 重构底层算子，引入词元级焦点损失，实现反向传播梯度精准聚焦长尾风格词元

### 4. 在线主机运行平台监测系统

- **项目简介**：基于 Flask 与混合存储的多主机资源监控系统，实现 CPU/内存/磁盘/网络指标实时采集、可视化及并发上报处理。
- **核心工作**：
  - 采用 Memcached + PyMySQL 混合存储，设计会话状态管理与 RBAC 权限分级
  - 完成多主机指标采集、ECharts 图表展示及高并发数据上报逻辑
