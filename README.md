## 🚀 核心项目 (Core Projects)

### 1. 智膳热鉴 —— 多模态大模型热量识别系统 (大创项目)
- **项目简介**：构建基于多模态大模型的健康饮食识别系统，结合视觉与自然语言处理技术，实现精准的食物热量评估。
- **核心工作**：
  - 设计并实现了核心的 **RAG（检索增强生成）模块**，优化垂直领域知识检索与大模型生成的结合链路。
  - 集成多模态交互能力，大幅提升系统的泛化能力与实际应用场景下的用户体验。

### 2. 基于 MindSpore 的 DeepSeek 模型垂直领域微调
- **项目简介**：在国产深度学习框架下，对 DeepSeek-R1-Distill-Qwen-1.5B 模型进行垂直场景的定制化训练与推理部署。
- **核心工作**：
  - 使用 **MindSpore** 及 **MindNLP** 框架，基于 `huanhuan-chat` 等特定数据集完成全流程微调。
  - 采用 **LoRA (Low-Rank Adaptation)** 参数高效微调技术，成功解决算力受限环境（如 AutoDL）下的模型收敛与推理效率问题。
- **源码入口**：[点击访问项目代码](#) *(注：替换为你的实际链接)*

### 3. 大数据底层基座实战与高分课程记录
- **项目简介**：涵盖分布式存储与高性能计算架构的底层逻辑实践。在底层大数据基座的深度实践与性能调优中展现了扎实的功底（核心课程 Hadoop 取得 97 分，Flink 取得 99 分）。
- **技术要点**：分布式文件系统架构原理、流式计算容错机制等。
